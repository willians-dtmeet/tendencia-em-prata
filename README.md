# Tendência em Prata - E-commerce de Joias

Um e-commerce moderno e responsivo para joias em prata, desenvolvido com Next.js 14, Tailwind CSS e Lucide React.

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide React

## Instalação

```bash
yarn install
```

## Desenvolvimento

```bash
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## Build

```bash
yarn build
yarn start
```
